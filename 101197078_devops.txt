COMP3104 Devops Course information

We will learn how to use BASH commands and how to to use github. We will learn how to configure git on local environment

-Version Control Systems
-SSH and Access Tokens
-VCS terminology
-CLI text editors 
-Leveraging VCS for DevOps
- Application Architectures,
- Monolith vs Microservice
architecture
- Vertical & horizontal scaling
- Tech stack behind reliable
applications
- Remote VCS repositories
- Creating remote repositories
- Git CLI

- DevOps culture and lifecycle
- Core DevOps principles
